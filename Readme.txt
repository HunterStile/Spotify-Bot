# Spotify Bot

## Descrizione
Il Spotify-Bot è un programma che automatizza alcune operazioni su Spotify, consentendo agli utenti di creare account, seguire playlist, e ascoltare brani.

## Funzionalità
- Creazione-Account : gli utenti possono creare in modo automatizzato gli account spotify
- Seguire Platlist : L'account puo seguire in modo automatizzato determinate playlist
- Ascoltare i brani: L'account puo ascoltare in modo automatizzato determinati brani.

## Requisiti di sistema
- Sistema operativo: Windows, macOS, Linux
- Python 3.x installato
- Dipendenze esterne: selenium,webdriver-manager,faker,pyautogui


## Installazione
1. Clona o scarica il repository del progetto.
2. Assicurati di avere Python 3.x installato sul tuo sistema.
3. Installa le dipendenze esterne eseguendo il comando `install.py`, situato nella cartella installazione.
